# Online Feedback System

## Overview
This is a simple web-based feedback system built using Flask and Python for the backend, XAMPP for MySQL database, and HTML/CSS for the frontend.

## Prerequisites
- Python 3.x
- XAMPP with MySQL
- Flask (install via `pip install Flask`)
- MySQL Connector (install via `pip install mysql-connector-python`)
- HTML, CSS for frontend
